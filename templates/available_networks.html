How to use the BBF

The following steps introduce a step-by-step procedure of the Blockchain Benchmarking Framework (BBF) usage. It is important to follow the step-guide and not skip any step. Each step will redirect you to the proper page where you will found more information for each functionality.

More details

  • Step 1 First you need to configure and start the monitoring engine. This step is one-time step and works for all the blockcchain networks. The following button will redirect you to the proper page where you should click the Start Button under the Monitoring tab. Start Monitoring Engine
  • Step 2 At this point the Monitoring engine is working and it is waiting to monitor the traffic in the blockchain network. Thus, it is important to configure and start a selected blockchain network. The following button will redirect you to the proper page where you will be able to configure your prefered blockchain. Configure a Blockchain Network
  • Step 3 At this stage you have to start the blockchain network you selected previously. It is important to start the blockchain network if you want to move to the next steps. You can configure and start more than two networks at the same time or your can stop one and the others will be still running. If you want to start or re-start the network go to the proper page by using the following button. Start a Blockchain Network
  • Step 4 We have developed our own traffic generator where the network generates random transactions in random generated accounts. You have to select a number of random accounts to be generated and number of transactions by using the following button. This will redirect you to the proper page. Start Traffic Generator
  • Step 5 This step is optional. In this step you can check the logs from the previous generated transaction. Keep the generated accounts and type them in the proper field in the Monitoring page by using the followin button. In the Monitoring page you will also fine the server_info functionality. Node / Public Key
  • Step 6 This step is optional. In this step you can check the grafana and docker logs by using the following button which will redirect you to the proper page. Logs / Grafana
  • Step 6a If you chosed to use the Grafan then you need to configure it. In order to configure it you have complete the following steps:
    • 1 2 3 4...
    Grafana Configuration
  • Step 7 In this step you can check the running networks and also get some more user-friendly information about the blockchain networks. Docker APIs
  • Step 8 You can stop any time any running network. You can stop it in case you do not need it anymore or if you want to free CPU and RAM. Follow the instructions under the STOP tab. Stop Network
  • Step 9 Stoping a network will enable you to restart it later, but this does not mean that it is removed from the disk. If you want to clean some space you need to clean the stopped network. Cleaning a network means it will need to re-configure it later. The following button will redirect you to proper page. Clean Network
  • Step 10 As final step you can stop the monitoring engine (stop Grafana, Dozzle etc.) and free the resources. Stop Monitoring
